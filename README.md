# OpenVPN Split Iplist

Интерактивный скрипт для настройки split tunneling в OpenVPN через iplist.opencck.org.

## Возможности

- split tunneling
- автоматическая генерация route push
- автоматическое обновление маршрутов
- NAT настройка
- systemd timer
- диагностика
- rollback

## Установка

```bash
sudo ./openvpn-split-iplist.sh
```

## Требования

- Ubuntu/Debian
- OpenVPN уже установлен и работает
- root доступ
